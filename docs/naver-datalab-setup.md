# 네이버 데이터랩 API 설정 가이드

트렌드 분석 기능을 사용하려면 네이버 데이터랩 API 키가 필요합니다.

## 1. 네이버 개발자 센터 등록

1. [네이버 개발자 센터](https://developers.naver.com/main/)에 접속합니다.
2. 네이버 계정으로 로그인합니다.
3. 상단 메뉴에서 **Application > 애플리케이션 등록**을 클릭합니다.

## 2. 애플리케이션 등록

1. **애플리케이션 이름**: `달래마켓 트렌드 분석` (원하는 이름 입력)
2. **사용 API**:
   - **검색** 섹션에서 **데이터랩(검색어 트렌드)** 선택
3. **비로그인 오픈 API 서비스 환경**:
   - **WEB 설정** 추가
   - **웹 서비스 URL**: `http://localhost:3000` (개발), `https://yourdomain.com` (프로덕션)
4. 등록 완료

## 3. API 인증 정보 확인

애플리케이션이 등록되면:
- **Client ID** (클라이언트 아이디)
- **Client Secret** (클라이언트 시크릿)

이 정보를 확인할 수 있습니다.

## 4. 환경변수 설정

`.env.local` 파일에 다음 내용을 추가하세요:

```bash
# 네이버 데이터랩 API 설정 (검색 트렌드 분석)
NAVER_DATALAB_CLIENT_ID=여기에_Client_ID를_입력
NAVER_DATALAB_CLIENT_SECRET=여기에_Client_Secret을_입력
```

## 5. 개발 서버 재시작

환경변수를 변경한 후 개발 서버를 재시작하세요:

```bash
# 개발 서버 중지 (Ctrl + C)
# 개발 서버 재시작
npm run dev
```

## 6. 기능 사용

1. 플랫폼 > 업무도구 페이지 접속
2. **트렌드 분석** 도구 선택
3. 검색 키워드 입력 (최대 5개)
4. 기간, 기기, 성별, 연령대 필터 설정
5. **트렌드 분석하기** 버튼 클릭

## API 사용량 제한

네이버 데이터랩 API는 다음과 같은 제한이 있습니다:
- **일일 호출 제한**: 1,000회
- **초당 호출 제한**: 10회

## 문제 해결

### API 키가 작동하지 않는 경우

1. Client ID와 Client Secret이 정확한지 확인
2. 네이버 개발자 센터에서 "데이터랩(검색어 트렌드)" API가 활성화되어 있는지 확인
3. 웹 서비스 URL이 올바르게 설정되어 있는지 확인
4. 개발 서버를 재시작했는지 확인

### 에러 메시지

- **"네이버 API 설정이 필요합니다"**: 환경변수가 설정되지 않았습니다.
- **"네이버 API 요청 실패"**: API 키가 잘못되었거나 호출 제한을 초과했습니다.
